# BUN Parser (CITS3007 Project)

A C project for **CITS3007** that parses and validates `.bun` files.

**Repository:** https://github.com/Farish-M/CITS3007-Project

## Requirements

### Main Executable

- A C compiler with C11 support, such as `gcc` or `clang`.
- `make` for building the parser.

### Other Programs

Python 3.8 or newer is required for running `bunfile_fixture_generator.py`.

#### `make test`

`check` and `pkg-config` for building and running the libcheck unit tests with `make test`.

On Ubuntu/Debian:

```sh
sudo apt install check pkg-config
```
